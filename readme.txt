== Description ==

This is a modified Hello Dolly plugin, created for educational purposes. When activated you will randomly see a lyric from <cite>Hello Darkness My Friend</cite> in the first paragraph of a post. You can comment the filter and uncomment hook to switch to “Hello Dolly” behavior.

In the future versions of the plugin, more hooks and filters will be added. So, students will be able to uncomment a hook/filter to test it’s behavior.
